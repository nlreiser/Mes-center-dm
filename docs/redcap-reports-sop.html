5 REDCap Reports SOP

This document functions as an instruction manual for creating a REDCap Report within the Data Request Database. Creating a report allows the user to easily view their or their teams in progress and completed Data Requests. The user can also create their own customized reports as needed.

For a more thorough guide for creating REDCap Reports in other study databases, view the following guide created by the University of Pennsylvania: REDCap Creating Reports_UPenn

5.1 Create a Mesulam Center Data Requests REDCap Report

  1. Navigate to the Data Request REDCap database.
  2. Under Applications, click ‘Data Exports, Reports, and Stats’.
  3. On the next page, click on the ‘+ Create New Report’ tab.
  4. Enter information about the New Report.
    1. Enter a Name of Report.
      1. Ex: All PPA Data Requests, Neuropath In-Progress, Nina-Completed
    2. Optional: Enter a brief Description of the Report.
  5. Select User Access.
    1. Select if All users or Custom users can view the report.
    2. Select if All users or Custom users can edit the report.
  6. Select all fields you would like to include in the report.
    1. Below is an example of common fields to include in reports that will provide adequate information about the data request and the progress made for each request.
  7. Select additional options for your report.
  8. Add filters to filter the report to specific staff or studies/departments that last worked on the data request.
    1. Add the filter for the variable ‘study_last’ and select which study to filter.
    2. To view a particular staff members’ open requests, add the filter for the variable ‘staff_last’ and select which staff member(s) to filter.
  9. Optional: Live Filters
    1. Live filters can be used after the report has been generated.
    2. Ex: Adding the variable ’data_request_complete allows the user to filter by complete statuses after the report has been generated.
  10. Optional: Determine your preferred ordering of the results.
    1. Ex: Sort the results first by priority (‘priority_score’) then status (‘rqst_ovstatus’).
  11. Click the Save Report button. A new screen will open up. Click View report to view.
  12. To view a report previously created, navigate to the ‘My Reports and Exports’ tab from the Reports Page. From here, you can view or export any reports. You can also navigate to the previously created reports from the sidebar.
